# 2026-1-CECD2-1-Runtime-5 Repository

This repository contains the consolidated work for the Runtime-5 project, organized into System implementation and Research experiments.

## Project Structure

### [System](./system)
The core implementation of the project.
- **[Frontend](./system/frontend):** React/Vite based user interface.
- **[Backend](./system/backend):** FastAPI backend providing the main pipeline and API services.

### [Research](./research)
Research and optimization experiments.
- **[Prompt Optimization](./research/prompt):** Experiments and datasets focused on prompt optimization.
- **[Cache Optimization](./research/cache):** Research on cache optimization and playbook storage.

## Getting Started
Each subdirectory contains its own documentation and setup instructions.
